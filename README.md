# Discord-Email-OTP-Bruteforce
Application to brute force Discord OTP codes for when you wanna change your email but lost the original email.

## Dependencies
This project depends on Firefox and Geckodriver, installation instructions can be seen below.
1. Linux

    Install geckodriver and firefox using your packagemanager
    
    `sudo pacman -S geckodriver firefox` for example
2. Windows
    Install [firefox](https://www.firefox.com/en-US/)

    Install [Geckodriver](https://stackoverflow.com/a/56926716)

## Running
1. Windows

    1.1 Download the exe from the [latest release](https://github.com/Slightly-Deranged-Lunatic/Discord-Email-OTP-Bruteforce/releases)

    1.2 Run the exe
2. Linux

    2.1 Download the executable file from the [latest release](https://github.com/Slightly-Deranged-Lunatic/Discord-Email-OTP-Bruteforce/releases)

    2.2 Ensure it has permission to run

    2.3 Run the file
